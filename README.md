# ShopSphere

Welcome to **ShopSphere** - *Where every click brings joy!*  
ShopSphere is a modern e-commerce platform built with React, providing an intuitive and dynamic shopping experience. The project features responsive design, smooth animations, user authentication, and a range of customizable components.

## Features

- **Dynamic Product Catalog**: Search, filter, and browse products by categories and keywords.
- **User Authentication**: Secure login, registration, and password recovery powered by `formik` and `yup`.
- **Shopping Cart & Wishlist**: Add items to the cart and wishlist with responsive, interactive UI feedback.
- **Promotional Discounts**: Redeemable promo codes for discounts on the total order.
- **Responsive Design**: Layouts built with Bootstrap for a seamless experience across devices.
- **Carousel & Animations**: Product and promotional carousels, hover effects, and interactive icons.
- **Notifications & Toasts**: User-friendly feedback and notifications for actions and events.

## Tech Stack & Packages

- **Frontend Framework**: React, React Router for navigation
- **State Management & Data Fetching**: React Query (`@tanstack/react-query`)
- **Form Handling & Validation**: Formik, Yup
- **UI Components & Icons**: Bootstrap, Bootstrap Icons, FontAwesome, React Icons
- **Image Carousels**: React Slick, Slick Carousel
- **APIs**: Axios for data fetching and API calls
- **Loading & Feedback**: React Loader Spinner, React Hot Toast, React Detect Offline
- **Other Tools**: React Helmet for head management, JWT Decode for token handling, AJV for JSON validation
